# MeetUpMe

개인 개발 포트폴리오 프로젝트  
소규모 취미 모임을 관리할 수 있는 웹 플랫폼입니다.

## 🔧 사용 기술
- Java 17 / Spring Boot
- MySQL
- JSP / Thymeleaf
- Bootstrap 5
- Spring Security

## 📌 주요 기능
- 회원가입 / 로그인 / 마이페이지
- 모임 생성 / 참여 / 후기 작성
- 관리자 페이지
- 후기 게시판 (Ajax 적용)

## 📂 프로젝트 구조
- controller: 요청 처리
- service: 비즈니스 로직
- repository: DB 연동
- templates/static: UI 구성
